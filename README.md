#### Steps to install & run on local system:

```sh
git clone https://github.com/iamkrvikash/Blog-App.git
cd Blog-App
npm i
npm update
node app.js
```

#### Next Open [http://localhost:3000/](http://localhost:3000/ "http://localhost:3000/")
